# Backend – Application E-commerce

Backend d’une application e-commerce développé en collaboration dans le cadre
du programme de mon stage chez **AIONE**.

Ce dépôt correspond à la partie serveur de l’application, distincte du frontend
mobile développé avec Flutter.

## Technologies
- Java
- Spring Boot
- Firebase (authentification et services)
- PostgreSQL (base de données relationnelle)
- Git

## Fonctionnalités principales
- API REST pour la gestion des utilisateurs et des produits
- Authentification et intégration avec Firebase
- Gestion des données via une base PostgreSQL
- Communication avec le frontend mobile


## Contributions
- Participation au développement de la logique backend
- Conception et implémentation d’API REST
- Intégration de Firebase avec Spring Boot
- Interaction avec la base de données PostgreSQL
- Travail collaboratif et gestion de versions avec Git



 Ce dépôt est une reprise personnelle du projet à des fins de présentation
académique. Le projet a été réalisé en équipe dans le cadre d’un stage.
